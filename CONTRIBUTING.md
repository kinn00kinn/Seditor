# Contributing

## Development flow

1. Install dependencies with `npm install`.
2. Start the desktop app with `npm run tauri dev`.
3. Run `npm run check` and `npm test` before submitting changes.

## Pull requests

- Keep changes focused and explain user-facing behavior changes clearly.
- Add or update tests when you touch logic that can be verified automatically.
- Do not mix unrelated refactors into the same pull request.

## Reporting bugs

Please include:

- operating system
- app version or commit hash
- reproduction steps
- expected behavior
- actual behavior
