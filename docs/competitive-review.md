# Competitive Review

## Similar OSS editors

Observed expectations from widely used Markdown editors such as MarkText, Zettlr, and ghostwriter:

- recent files / reopen flow
- session restore
- status information such as word count
- release automation and contribution documentation

## Microsoft Notepad comparison

Public Windows Insider notes indicate that modern Notepad added:

- automatic session state saving
- tab restore
- unsaved content restore across sessions

Seditor now matches part of that direction with:

- startup restore of the last opened file
- auto save
- recent file reopening

Still missing compared with modern Notepad:

- multi-tab editing
- richer plain-text focused editing features such as spellcheck or autocorrect
- stronger session model for multiple open documents

## Follow-up candidates

- multi-document tabs
- spellcheck integration
- split preview / editor mode
- export presets
